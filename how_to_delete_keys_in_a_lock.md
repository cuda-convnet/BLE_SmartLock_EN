# 刪除鎖具內的鑰匙

具有**管理者**鑰匙的使用者，在鎖具設定選取**管理裝置內的鑰匙**，該裝置內的鑰匙會依序列出。

選取鑰匙向左滑動，點選**刪除**圖示以刪除鎖具內的鑰匙。
